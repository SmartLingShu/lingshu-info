### Tab说明

| Tab | 说明 |
|-----|------|
| 失物寻找 | 用户丢失物品，寻求帮助 |
| 拾物招领 | 用户捡到物品，寻找失主 |
| 我发布的 | 当前用户发布的信息 |

### 物品状态

| 状态 | 颜色 | 说明 |
|------|------|------|
| 寻找中 | 🔴 红色 | 失物正在寻找 |
| 待认领 | 🟡 橙色 | 拾物等待认领 |
| 已找到 | 🟢 绿色 | 失物已找回 |
| 已认领 | 🟢 绿色 | 拾物已被认领 |
| 已过期 | ⚪ 灰色 | 超过30天未处理 |

## 11.6 接口依赖

| 接口名称 | 请求方式 | 接口路径 | 说明 |
|----------|----------|----------|------|
| 失物列表 | GET | `/api/v1/lost-found/list` | 获取失物招领列表 |
| 发布信息 | POST | `/api/v1/lost-found/publish` | 发布失物/拾物 |
| 详情 | GET | `/api/v1/lost-found/{id}` | 获取详情 |
| 申请认领 | POST | `/api/v1/lost-found/claim` | 申请认领 |
| 确认找到 | POST | `/api/v1/lost-found/found` | 确认已找到 |
| 确认认领 | POST | `/api/v1/lost-found/claimed` | 确认已认领 |
