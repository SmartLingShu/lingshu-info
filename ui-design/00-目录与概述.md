## 🎯 项目背景

**智习灵枢**是一款面向高校和图书馆的智能自习室座位预约管理系统，小程序端作为用户主要使用入口，需要提供：

- **便捷的座位预约体验** - 快速查找、选座、预约
- **高效的签到签退流程** - 扫码即签，操作简单
- **丰富的学习激励机制** - 积分、成就、排行榜
- **智能的AI辅助功能** - 对话预约、智能推荐

## 🔄 核心用户流程

### 流程1：首次使用流程

```
启动页 → 登录授权页 → 同意协议 → 首页
```

### 流程2：座位预约流程

```
首页/预约页 → 选择教学楼 → 选择楼层 → 选择教室 → 查看座位图 → 选择座位 → 选择时段 → 确认预约 → 预约成功
```

### 流程3：签到签退流程

```
扫码页 → 扫描座位二维码 → 识别座位信息 → 确认签到/签退 → 操作成功
```

### 流程4：AI智能预约流程（规划中）

```
发现页 → AI助手 → 语音/文字输入需求 → AI推荐座位 → 一键预约 → 预约成功
```
