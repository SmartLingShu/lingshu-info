### 页面元素

| 元素 | 说明 | 样式 |
| ---- | ---- | ---- |
| Logo | 系统Logo | 居中，宽度120px |

| 记住我 | 复选框 | 左对齐 |
| 登录按钮 | 提交登录 | 主色调，宽度100% |

### 表单验证规则

| 字段 | 规则 | 错误提示 |
| ---- | ---- | -------- |
| 用户名 | 必填，4-20位字符 | "请输入用户名" |
| 密码 | 必填，6-20位字符 | "请输入密码" |
| 验证码 | 必填，4位字符 | "请输入验证码" |

### 交互说明

1. **登录流程**：
   - 输入用户名、密码、验证码
   - 点击登录按钮
   - 验证成功 → 跳转工作台
   - 验证失败 → 显示错误信息

## 1.4 接口依赖

| 接口名称 | 请求方式 | 接口路径 | 说明 |
| -------- | -------- | -------- | ---- |
| 获取验证码 | GET | `/admin-api/system/captcha/get` | 获取图形验证码 |
| 登录 | POST | `/admin-api/system/auth/login` | 账号密码登录 |
| 获取用户信息 | GET | `/admin-api/system/auth/get-permission-info` | 获取当前用户信息和权限 |
| 退出登录 | POST | `/admin-api/system/auth/logout` | 退出登录 |
| 修改密码 | PUT | `/admin-api/system/user/profile/update-password` | 修改密码 |
| 工作台统计 | GET | `/admin-api/study/dashboard/statistics` | 获取工作台统计数据 |
| 预约趋势 | GET | `/admin-api/study/dashboard/reservation-trend` | 获取预约趋势数据 |
| 教室使用率排行 | GET | `/admin-api/study/dashboard/classroom-rank` | 获取教室使用率排行 |
