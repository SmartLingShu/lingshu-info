## 🎯 项目背景

**智习灵枢后台管理系统**是面向管理员的Web端管理平台，用于管理自习室资源、处理预约业务、监控运营数据。主要功能包括：

- **自习室资源管理** - 教学楼、楼层、教室、座位的增删改查
- **预约业务管理** - 预约记录查询、签到管理、违约处理
- **用户管理** - 小程序用户管理、信用积分管理、黑名单管理
- **运营数据分析** - 使用率统计、学习时长分析、报表导出
- **系统配置** - 预约规则、积分规则、系统参数设置

## 🎨 设计规范

### 技术栈

| 技术 | 版本 | 用途 |
| ---- | ---- | ---- |
| Vue | 3.x | 核心框架 |
| TypeScript | 5.x | 开发语言 |
| Element Plus | 2.x | UI组件库 |
| ECharts | 5.x | 数据可视化 |

### 品牌色彩

| 色彩类型 | 色值 | 使用场景 |
| -------- | ---- | -------- |
| 主色调 | #409EFF | 主按钮、导航选中、重点强调 |
| 成功色 | #67C23A | 成功状态、可用状态 |
| 警告色 | #E6A23C | 警告提示、待处理状态 |
| 危险色 | #F56C6C | 错误提示、删除操作 |
| 信息色 | #909399 | 次要文字、禁用状态 |
| 背景色 | #F5F7FA | 页面背景 |

### 布局规范

| 区域 | 宽度 | 说明 |
| ---- | ---- | ---- |
| 侧边栏 | 200px | 收起时64px |
| 顶部导航 | 60px | 固定高度 |
| 内容区域 | 自适应 | 最小宽度1200px |
| 页面边距 | 20px | 内容区域内边距 |

### 表格规范

| 元素 | 规范 |
| ---- | ---- |
| 表头 | 背景色#FAFAFA，字体加粗 |
| 行高 | 默认48px |
| 分页 | 默认每页20条，可选10/20/50/100 |
| 操作列 | 固定在右侧，宽度根据按钮数量自适应 |

### 表单规范

| 元素 | 规范 |
| ---- | ---- |
| 标签宽度 | 统一100px，右对齐 |
| 输入框宽度 | 默认240px，长文本400px |
| 必填标识 | 红色星号在标签前 |
| 校验提示 | 输入框下方红色文字 |

## 🔄 核心业务流程

### 流程1：自习室资源配置

```text
创建教学楼 → 创建楼层 → 创建教室 → 配置座位布局 → 发布上线
```

### 流程2：预约违约处理

```text
系统检测违约 → 生成违约记录 → 扣除信用分 → 达到阈值加入黑名单
```

### 流程3：公告发布

```text
编辑公告内容 → 选择公告等级 → 设置发布时间 → 发布/定时发布
```
